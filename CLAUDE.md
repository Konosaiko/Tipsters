# Project Context

This is a learning-first SaaS project: a platform for sports tipsters and their subscribers.

The goal is to build a clean, understandable, modern web application without over-engineering.

---

## Tech Stack (Fixed)

- Backend: Node.js + TypeScript + Express.js
- Database: PostgreSQL
- ORM: Prisma
- Frontend: Vite + React (later)
- API-first architecture

---

## Core Principles

- Keep things simple and explicit
- Avoid unnecessary abstractions
- Prefer clarity over cleverness
- Optimize for learning and maintainability

---

## Workflow Rules

- Always start in plan mode
- Never write code without an agreed plan
- One feature at a time
- Ask questions when requirements are unclear

---

## Constraints

- Monolithic application
- No microservices
- No complex frameworks or hidden magic
