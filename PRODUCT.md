# Product Overview

This project is a SaaS platform for sports tipsters and their subscribers.

The platform allows tipsters to publish betting tips and users to follow or subscribe to them in a transparent and simple way.

The primary goal is learning: building a clean, understandable backend system without over-engineering.

---

## Target Users

### Tipsters
- Want to publish betting tips
- Want to build credibility over time
- May offer free or paid tips

### Subscribers
- Want to follow tipsters
- Want to see past performance
- Want a simple and trustworthy interface

---

## Core Principles

- Simplicity over feature richness
- Transparency over marketing tricks
- Clarity over clever abstractions

---

## MVP Scope

### Must Have
- User authentication
- Tipster profile
- Ability for a tipster to publish a tip
- Ability for users to view tips
- Basic association between users and tipsters

### Nice to Have (Later)
- Paid subscriptions
- Performance statistics
- Comments or likes
- Notifications
- Admin moderation

---

## Explicit Non-Goals (For Now)

- Advanced betting analytics
- Complex recommendation algorithms
- Mobile application
- Real-time odds integration
